## Installation

### Requirements

- Node.js (recommended LTS version)
- npm or yarn

### Steps

Clone the repository to your local machine:

- git clone https://github.com/your-username/your-repo.git
- cd your-repo

Install dependencies:

  - npm install

Start the application:

  - npm run dev
